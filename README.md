## 📊 Yahoo Finance Stock Dashboard

Fetches and analyzes historical stock data using the `yfinance` API.

- 📥 Pulls 1 year of historical data for a chosen ticker (e.g., AAPL)
- 🧼 Cleans missing values using forward fill, backward fill, and interpolation
- 📊 Visualizes stock trends using price, volume, and moving averages
- 💾 Saves cleaned dataset to CSV for dashboard use
- ✅ Great starting point for financial dashboards or time series modeling
